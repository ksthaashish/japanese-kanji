Quick Add Multiple Kanji

Enter one kanji per line using this format (category is optional):

character; meaning; onyomi (comma); kunyomi (comma); category; example1|hiragana1|english1, example2|hiragana2|english2

If category is omitted, it defaults to "user".

Example:
猫; cat; ビョウ,ミョウ; ねこ; jlpt-n5; 猫|ねこ|cat, 子猫|こねこ|kitten
